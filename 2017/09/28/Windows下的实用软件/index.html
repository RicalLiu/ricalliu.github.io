Windows 系统下有许多有趣实用的软件。

  • PotPlayer 是一款解码能力强大的本地播放器。除了播放迅雷缓存 文件外，还可以逐帧播放 、加上完美的快进、快退调速功能。内存小 和界面美观 是看家本领。还可以通过导入源收看电视直播。总之，他是完美之选。
  • Bandzip 是一款免费的压缩软件。没有任何广告和捆绑。简洁的界面和全面的功能让我直接抛弃了7z 。
  • Everything 完虐Windows系统自带搜索功能的超强搜索软件。无需多言，就是强！
  • Free Download Manager ：当你受够了迅雷的臃肿和广告之后，又不愿意选择IDM，那么FDM就是最好的选择。简洁、高效，支持各种格式的下载链接。支持P2P下载技术……简约不简单！
  • Dism++ ，高效、完美的开源Windows优化软件。小巧而又整洁，他的优势务虚多言。尤其是老系统 优化得心应手。
  • Geek uninstaller 免费版本就已经足够强大的卸载软件，帮助你清理软件遗留痕迹。
  • Rufus ，一款不接受捐赠 的绿色开源 创建启动盘的软件。就是这么任性！
  • Tsearch ，一款整合网络资源的资源搜索软件。好用加全面！而且纯绿色！
  • Cmder ,Windows平台下相当好用的一款终端操作软件。
  • Listen 1 ：整合QQ、虾米和网易云播放器资源的GIthub项目。支持chrome和Firefox扩展。三大桌面平台通用。
  • SublimeText ，有趣使用的文本编辑器。加上插件之后就是完美的代码编辑器。还可以查看图片！
  • Typora 完美的Markdown编辑器。
  • ShadowsocksR ,超越VPN的翻墙神器。记得找服务供应商购买服务。
