Ubuntu 搭建个人博客 

刚为自己的笔记本装上双系统，Win10 + Ubuntu 16.04 LTS 64bit，看见知乎上有人发布了一篇使用Win10搭建个人博客的[文章](https://zhuanlan.zhihu.com/p/25471760) 。感觉很有意思，所以想试试自己用Ubuntu搭建一个。

搭建环境：Ubuntu 16.04 LTS 64bit

第一步：注册[github](https://github.com/) 账号，建立新的repository 命名为 yourname.github.io；

第二步：安装git和nodejs；

第三步：安装npm并升级nodejs；

第四步：配置git,并创建RSA私钥和公钥。在主目录下查看.ssh/id_rsa.pub文件，复制内容至github->settings->SSH and GPG keys->SSh keys->new ssh key(更多git使用方法[戳这里](http://www.liaoxuefeng.com/wiki/0013739516305929606dd18361248578c67b8067c8c017b000) );

第五步：安装hexo；

第六步：新建目标目录并初始化hexo；

第七步：试用hexo个人博客；

第八步：用markdown[markdown](http://wowubuntu.com/markdown/basic.html) 写自己的博客吧！

第九步：推向云端！

第十步：应该是没有了……当然，你要访问一下试试看
